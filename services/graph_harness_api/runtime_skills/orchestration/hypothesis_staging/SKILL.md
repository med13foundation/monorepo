summary: Stage speculative but evidence-backed hypotheses without overstating certainty.
tools: []
---
Use this skill when the output should be a candidate hypothesis rather than an asserted fact.

Keep hypotheses concrete, falsifiable, and traceable to the graph context that motivated them.
Separate supporting observations from open questions and unresolved assumptions.

If the evidence is too weak for even a staged hypothesis, return a fallback decision and say
what additional information is needed.
