This workflow may discuss possible graph writes, but discussion is not approval.

Keep chat answers grounded first.
Only surface graph-write candidates when the answer is verified or explicitly marked as needing
review, and explain the review status clearly.

Prefer proposal language over imperative write language in conversational flows.
