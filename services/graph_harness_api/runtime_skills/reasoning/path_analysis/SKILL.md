Use reasoning paths to identify convergence, not just connectivity.

Prioritize paths with stronger confidence, clearer evidence bundles, and repeated support
across independent routes.
Inspect detailed path payloads before proposing a mechanistic story.

If a mechanism looks plausible but still speculative, stage it as a hypothesis instead of
presenting it as established knowledge.
