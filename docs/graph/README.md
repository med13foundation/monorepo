# Graph Docs

This folder is the consolidated documentation set for the implemented kernel
graph architecture.

Files:

1. `architecture.md`
   The implemented graph model, data flow, write/read ownership, and invariants.
2. `endpoints.md`
   The graph-related API surface grouped by entities, observations, relations,
   claims, graph documents, graph search, and graph connection workflows.
3. `use-cases.md`
   The primary user and system workflows that operate on the graph.
4. `examples.md`
   Concrete examples of claim materialization, claim-only disagreement,
   graph-document output, and readiness checks.

This section is meant to be the fastest path for understanding how the graph
works today without reading proposal or migration history first.
